# NexoBol - Plataforma P2P de Intercambio

Este es el repositorio oficial del prototipo de alta fidelidad de **NexoBol**, la plataforma de intercambio P2P líder en Bolivia.

## Características

*   **Identidad Visual**: Diseño profesional en modo oscuro (Midnight Blue) con acentos en Cian Eléctrico.
*   **Mercado P2P**: Simulación funcional de compra y venta de USDT con filtros dinámicos.
*   **Dashboard**: Panel de usuario con gráficos y resumen de activos.
*   **Seguridad**: Validación de formularios y estructura robusta en React + TypeScript.

## Tecnologías

*   React 18
*   TypeScript
*   Vite
*   CSS Variables (Design System personalizado)
*   Lucide React (Iconografía)

## Instalación Local

1.  Clonar el repositorio.
2.  Instalar dependencias: `npm install`
3.  Correr servidor de desarrollo: `npm run dev`

## Despliegue

Este proyecto está listo para ser desplegado en **Vercel** o **Netlify**.
